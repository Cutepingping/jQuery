The Single Responsibility Principle（单一职责SRP）<br/>

The Liskov Substitution Principle（里氏替换原则LSP）<br/>
The Interface Segregation Principle（接口分离原则ISP）<br/>
The Dependency Inversion Principle（依赖反转原则DIP）<br/>
